# 🏥 Village Health Voice
### AI-powered Rural Health Assistant using Gemma 4

![Language](https://img.shields.io/badge/Multilingual-Tamil%20%7C%20Hindi%20%7C%20English-green)
![Track](https://img.shields.io/badge/Track-Health%20%26%20Sciences-red)

## 🌍 The Problem
Over 600 million people in rural India lack easy access to doctors. 
A simple fever can become fatal when the nearest clinic is hours away 
and the patient doesn't know if it's serious or not.

## 💡 The Solution
**Village Health Voice** is an offline-capable AI health triage assistant 
powered by **Gemma 4**. It helps rural communities and community health 
workers quickly assess symptoms and decide:
- 🚨 **EMERGENCY** — Go to hospital immediately
- ⚠️ **MODERATE** — Visit a clinic soon  
- ✅ **MILD** — Safe to treat at home

## ✨ Features
- 🤖 Powered by **Gemma 4** (gemma-4-31b-it)
- 🌐 **Multilingual** — Tamil, Hindi, Telugu, English and 140+ languages
- 📱 Works on basic smartphones
- 🎯 Smart triage with color-coded severity
- 🏠 Practical home remedy suggestions
- ⚕️ Clear emergency warnings

## 🛠️ Tech Stack
- **AI Model:** Gemma 4 (gemma-4-31b-it) via Google AI Studio
- **Language:** Python
- **Interface:** Jupyter Notebook with ipywidgets
- **Platform:** Kaggle

## 🚀 How to Run
1. Open the Kaggle notebook
2. Add your Google AI Studio API key as a Kaggle Secret named `GOOGLE_API_KEY`
3. Enable internet in Session options
4. Run all cells
5. Type symptoms in your language and click Check Symptoms
